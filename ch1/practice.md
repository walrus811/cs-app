#### B.

S = 1.67x
T_n = T_o / S = 25 / 1.67 = 15h

S = 1 / {(1 - 0.6) + 0.6/k}
1.67 = 1 / {0.4 + 0.6/k}
0.4 + 0.6/k = 1 / 1.67
0.6/k = 1 / 1.67 - 0.4
k = 0.6 / (1 / 1.67 - 0.4) = 3

The new speed is 3 \* 100 = `300km/h`
